Shell 模式匹配

Shell 模式匹配跟正则表达式有点不同,只能用来做简单的字符串匹配.

  • * 匹配 0 或多个字符
  • ? 匹配任意一个字符
  • [string] 匹配[]中的字符,例如说[a-z0-9]匹配小写字符和数字,可以在加入!或^来反向选择,例如说[^a-z]匹配非小写字母的字符.
  • \ 转义符
